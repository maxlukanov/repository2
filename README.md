Формула для `arange` може бути наступною: `start + i * step`, де `start` - початкове значення, `i` - індекс елементу у послідовності, `step` - крок.

   Формули для `arange` та `linspace` використовуються у різних випадках. `arange` підходить для генерації послідовності з рівномірним кроком, тоді як `linspace` дозволяє
